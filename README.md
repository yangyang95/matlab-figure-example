# Matlab 繪圖範例
### Example 1.
範例 1 教學如何利用 matlab 程式碼製作圖形，將 y =  sin(x) 與 y = cos(x) 的圖形繪製於同一張圖形之上。<br>
同時，此範例教學使用程式碼完成圖形的細節設定，包含：線條粗系、顏色、標題、x, y 軸標籤等等設定。
